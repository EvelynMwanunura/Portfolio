Commemorative Days Calendar

I built a commemorative calendar that displays special days from a JSON file. Using JavaScript, I calculated the correct calendar date and day of the week for each event. I also added functionality to generate downloadable .ics calendar files, allowing users to add events to their personal calendars. The events were dynamically rendered on a styled, interactive calendar interface.

Codewars Leaderboard

A dynamic web app built with JavaScript, HTML, and CSS that compares multiple Codewars users' stats in real time. Users can enter multiple usernames (comma-separated), and the app uses async/await with Promise.all to fetch data efficiently from the Codewars API. It displays a leaderboard showing each user’s rank, score, and other key details for easy comparison.

Live Demo GitHub

Spaced Repetition Tracker

A collaborative study tool built with JavaScript and HTML (no CSS) for four users. The app allows each user to input study topics and uses custom logic to calculate spaced repetition dates (e.g. Day 1, 3, 7, etc.) to optimize memory retention. All data is stored in local storage, so users can track their revision schedules directly in the browser without needing a backend.

Live Demo GitHub

Weather App

A responsive weather application that displays the current weather, wind speed, humidity, and a 5-day forecast for any searched city. Built using JavaScript, HTML, and CSS, it fetches real-time data from a weather API to provide users with accurate and up-to-date weather information.
